# # Web TRPG Homebrew Arknights

A fan-made tabletop RPG adaptation inspired by the world of Arknights.

![Banner](./public/banner.png)

## About

**Web TRPG Homebrew Arknights** is a browser-based tabletop RPG project designed to bring the world of Arknights into a roleplaying experience.

Players can create Operators, explore Terra, complete missions, battle dangerous threats, and develop their stories through a custom TRPG system inspired by the setting, factions, and themes of Arknights.

This project is created for educational and fan-content purposes only and is not affiliated with Hypergryph, Studio Montagne, Yostar, or any official Arknights-related organizations.

## Installation

Clone the repository:

```bash
git clone https://github.com/GhostFak/Web-TRPG-Homebrew-Arknights.git
```

Navigate to the project:

```bash
cd Web-TRPG-Homebrew-Arknights
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```
